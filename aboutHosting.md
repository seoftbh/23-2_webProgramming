# PHP 호스팅 서버에 대한 정보
## PHP 웹 서버 구현 방법
- LEMP Stack 사용

> [!TIP]
> **LEMP**
> Linux + NginX + MariaDB + PHP
>
> ---
> **L**: Linux -> OS
> - 서버 운영을 위한 기본 운영 체제
>
> **E**: NginX(EnginX) -> Web Server
> - 빠르고 경량의 웹 서버로, 정적 파일 제공, 로드 밸런싱, 프록시 서버 등의 기능을 수행
>
> **M**: MariaDB -> RDBMS
> - 관계형 데이터베이스 시스템으로, 웹 애플리케이션에서 데이터를 저장하고 관리하는 데 사용
>
> **P**: Web Page
> - 서버 측 스크립팅 언어
> - 동적 웹 페이지를 생성하기 위해 사용되는 서버 측 스크립팅 언어
> - 데이터베이스와 상호 작용
