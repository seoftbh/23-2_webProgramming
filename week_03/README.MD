* 구현 기능
    * 셀렉터

            * 태그 이름을 콤마로 구분하여 여러 태그에 같은 스타일 지정
        * 자식 셀렉터
            * `>`
        * 자손 셀렉터
            * ` `(공백으로 구분)
        * class 셀렉터
            `.`
        * id 셀렉터
            `#`
        * 가상 클래스 셀렉터
            * `:first-letter` -> 제목(`Web Programming`)의 맨 첫 글자만 색상 변경
            * `:hover` -> 마우스 이동 시 배경 색상이 바뀜
